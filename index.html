Welcome to CWAG Bot

Your all-purpose bot for moderation, games, and more!

List of Commands:

  • !mute @user [time] - Mute a user for a specified time (in seconds).
  • !unmute @user - Unmute a user.
  • !kick @user - Kick a user from the server.
  • !ban @user - Ban a user from the server.
  • !clear [amount] - Clear a number of messages from the channel.
  • !lock - Lock the current channel (prevents users from sending messages).
  • !unlock - Unlock the current channel (allows users to send messages again).
  • !guess - Start a number guessing game.
